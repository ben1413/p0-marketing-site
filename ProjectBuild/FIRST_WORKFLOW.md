# FIRST_WORKFLOW

## Goal

Demonstrate a full collaboration loop between humans, agents, and the ledger.

## Example Scenario

A user creates a project named `Community Clubhouse`.

The system creates a room for collaboration.

The user creates a track named `Sponsorships`.

Inside the track the user asks the Research Agent:

```text
Identify potential sponsors.
```

Core executes the agent.

The agent proposes a task:

```text
Research local gaming companies.
```

The task appears on the project board.

The user reviews results and selects a sponsor.

The decision is promoted to the ledger.

Ledger entry contains:

- `summary`
- `actor`
- `authorityMode`
- `timestamp`

## Outcome

The project now has a permanent decision record.

The full lifecycle has been demonstrated.
