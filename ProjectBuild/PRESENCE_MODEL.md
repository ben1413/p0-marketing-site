# PRESENCE_MODEL

## Presence Model

ProjectBuild supports collaborative work between humans and agents.

## Participants

- Humans
- Agents
- External systems

Presence is scoped to a project room.

Example participant list:

- Ben
- Sarah
- Research Agent
- Legal Agent
- Finance Agent

## Presence States

- `active`
- `idle`
- `agent-running`
- `agent-paused`

Agents appear as participants in the room rather than external tools.

This design reinforces the concept of a multi-human, multi-agent workforce.
