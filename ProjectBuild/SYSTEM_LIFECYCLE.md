# SYSTEM_LIFECYCLE

## Project0 System Lifecycle

All work inside Project0 follows a consistent lifecycle.

### Intent

A human or system defines an objective.

### Run

Agents execute work through the Core runtime.

### Evaluate

Agent outputs can be assessed through evaluation mechanisms.

### Promote

Important outcomes are explicitly promoted.

### Ledger

Promoted items become immutable records.

## Lifecycle Flow

```text
Intent
  ->
Run
  ->
Evaluate
  ->
Promote
  ->
Ledger
```

Exploratory work remains mutable.

Only promoted outcomes become part of permanent system history.
