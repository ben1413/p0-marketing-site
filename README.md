# Project0.io — Marketing Website

Marketing site for **Project0** — infrastructure for the multi-human, multi-agent workforce.

## Quick Start

**Option 1: Open directly**
```bash
open index.html
```

**Option 2: Serve locally (recommended for fonts/CORS)**
```bash
npx serve .
# or
python3 -m http.server 8080
```

Then visit `http://localhost:3000` (serve) or `http://localhost:8080` (python).

## Deploy (Vercel)

```bash
cd project0-marketing
vercel
```

Or connect the repo in the [Vercel dashboard](https://vercel.com) — it will detect the static site automatically.
