WHAT IT IS

The backend system that makes
AI work accountable.

AGENT RUNTIME

Run single and multi-agent workflows on shared infrastructure. Core handles orchestration, cognitive preload, memory context, action gating, and run records.

GOVERNANCE GATE

Every action passes through deterministic policy enforcement. Permissions, restrictions, human acknowledgment rules - enforced and recorded.

EVALUATION ENGINE

Create evaluations, close decision windows, run agent assessments, and review results. Built for real operational oversight, not demo environments.

AUTHORITY + CONTROL

Make authority explicit. Delegate agent authority, require human oversight where needed, and halt, suspend, or resume agents when behavior crosses a boundary.

LEDGER + PROMOTE

Move from mutable work into durable history. Promote records the commitment boundary with actor attribution, authority mode, integrity metadata, and truth posture.

ARTIFACTS + AUDIT

Generate sealed artifacts for closed evaluations. Export audit trails in a format built for review, defensibility, and downstream trust.

THE PRIMITIVES

Every surface inherits these.
Call them directly if you're building your own.

run()

Execute an agent with governed context. Every run is persisted. Every trace is attributable.

POST /api/v1/agents/run

promote()

Commit a decision across the governance boundary. The line between thinking and deciding.

POST /api/v1/ledger/promote

create_evaluation()

Open a bounded decision window.

POST /api/v1/evaluations

close_evaluation()

Close the decision window. Ready to seal.

POST /api/v1/evaluations/:id/close

delegate()

Grant authority to an agent with explicit scope. Duration recorded. Delegation traceable.

POST /api/v1/authority/delegations

halt()

Stop execution when governance rules trigger.

POST /api/v1/agents/:id/halt

suspend()

Suspend future runs for an agent.

POST /api/v1/agents/:id/suspend

resume()

Re-enable runs for a suspended agent.

POST /api/v1/agents/:id/resume

generate_artifact()

Seal a closed evaluation into Artifact 5.

POST /api/v1/artifacts/generate

export_audit()

Export a structured, independently verifiable audit trail for any evaluation.

POST /api/v1/audit/export
